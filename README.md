## Installation Windows
Download:
https://www.openscad.org/downloads.html

as stated here https://en.wikibooks.org/wiki/OpenSCAD_User_Manual/Libraries
go to File->Show Library Folder...
copy or clone the whole git repo there

restart OpenSCAD 
load cap.scad
